# Python Security Tools

This repository contains simple Python scripts designed to practice cybersecurity automation and defensive security techniques.

The goal of these scripts is to demonstrate how automation can assist security analysts in identifying suspicious activity and improving investigation workflows.

## Objectives

• Learn how Python can assist cybersecurity tasks  
• Automate repetitive security analysis steps  
• Practice reading and analyzing system data  
• Strengthen technical problem-solving skills

## Planned Tools

### Log Parser
A Python script designed to analyze log files and highlight suspicious entries.

### Port Scanner
A simple scanner to detect open ports on a target system for educational purposes.

### IP Activity Monitor
Script that helps identify unusual IP activity patterns.

### Security Automation Scripts
Small utilities that demonstrate how Python can assist security monitoring tasks.

## Skills Practiced

• Python scripting  
• Automation thinking  
• Security analysis workflow  
• Problem solving and debugging

This repository demonstrates my growing ability to combine cybersecurity knowledge with programming to automate defensive security tasks.
